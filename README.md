# @syberisle/postcss-at-apply

[PostCSS]() plugin that allows you to use `@apply` similar to Tailwind CSS.

# Installation

```shell
npm install --save-dev @syberisle/postcss-at-apply
```

Add to your plugins list
```javascript
module.exports = {
	plugins: {
		"@syberisle/postcss-at-apply": {}
    }
}
```
